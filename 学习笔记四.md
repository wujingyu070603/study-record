### 2.bs4，requesets

bs4可以用于在乱七八糟的网页中找到需要的东西。

+ res.text:网页的源代码
+ find_all（）：“帮我找出所有..."（返回列表）
+ find（）：找出第一个
+ get_text ：只得到文字不得到标签

resquests用于得到网页上的东西。

~~~python
from bs4 import BeautifulSoup
import requests
from IPython.display import HTML,display
url = "https://httpbin.org/html"
res = requests.get(url) #获取这个网址的内容
soup = BeautifulSoup(res.text,"html.parser") #res.text指的是获取的网址源代码，html.parser是HTML解析器，用来帮助程序看懂网页结构
all_text = soup.find_all("p")#找到带有“p“这个标签的段落
combined_text = "" #创建一个空的字符串
for text in all_text:
    combined_text = combined_text + "\n" + text.get_text() #把所有的段落拼起来
print(combined_text)
~~~

### 3. 运用API在openweather网站获取天气信息

登录网站获取api key：

![image-20260514171109499](C:\Users\35641\AppData\Roaming\Typora\typora-user-images\image-20260514171109499.png)

写获取信息的代码：

~~~python

import requests #请求运用requests
api_key= "c1988358479cd2824db91812f40286c1" 
city="Xi'an"
url=f"http://api.openweathermap.org/data/2.5/weather?q={city}&units=metric&appid={api_key}"

#api.openweathermap.org：天气 API 服务器地址
#/data/2.5/weather：获取 “当前天气” 的接口路径
#q={city}：城市参数（Xi'an）
#units=metric：单位设置 → 摄氏度
#appid={api_key}：把你的 API Key 填进去，验证身份
res=requests.get(url)
data = res.json() #返回为json格式
"""
json 是这样，就是字典的样子
{
  "weather": [{"description": "clear sky"}],
  "main": {"temp": 25}
}
"""
print(f"天气状况：{data['weather'][0]['description']}")
print(f"温度：{data['main']['temp']}摄氏度")
    
~~~
